Findr.LBeta — Type
LBeta(α,β)

The LBeta distribution with parameters $\alpha$ and $\beta$ is defined as the distribution of a random variable

$X=-\frac{1}{2}(\ln(1-Y))$

where $Y\sim\operatorname{Beta}(\alpha, \beta)$

Findr.llrstats_row — Method
llrstats_row(Y,E,row)

Compute the sufficient statistics to compute the log-likelihood ratios for Findr tests 2-5 for a given row (gene) of gene expression matrix Y with categorical instrument E against all other rows of Y.

Y is assumed to have undergone supernormalization with each row having mean zero and variance one. The LLRs are scaled by the number of samples.

The sufficient statistics are:

  • the group sizes (number of elements) for each group (unique value) in E
  • the mean of each row of matrix Y for each of the groups (unique values) in E
  • the weighted average variance of each row of matrix Y over the groups (unique values) in E
  • the weighted average covariance between the given row of Y and all other rows of Y over the groups (unique values) of E

Findr.supernormalize — Function
supernormalize(X[, c])

Convert each row of matrix X of reals into standard normally distributed values using a rank-based inverse normal transformation. Then scale each row to have variance one.

Note that after the inverse normal transformation, each row has mean zero and identical variance (if we use ordinal ranking). Hence rescaling can be done once on the whole matrix.

The formula and default value for the paramater c come from this paper: https://www.ncbi.nlm.nih.gov/pmc/articles/PMC2921808/
